# Ícones PWA

Para gerar os ícones PNG necessários para o PWA, você pode usar uma das seguintes ferramentas online:

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
2. **Favicon Generator**: https://realfavicongenerator.net/

## Ícones Necessários

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Processo

1. Use o arquivo `favicon.svg` existente como base
2. Gere todos os tamanhos acima
3. Coloque os arquivos PNG nesta pasta
4. O PWA estará completo e instalável
